除以一个数等于乘上这个数的倒数，在除法取余的情况下，就是乘上这个数的逆元

除以一个数再取模等同于乘以这个数的逆元再取模
什么意思? 设 𝑖𝑛𝑣[𝑏] 是 𝑏 的逆元, 那么 (𝑎𝑏)%𝑝=(𝑎∗𝑖𝑛𝑣[𝑏])%𝑝

(ab)%p = (a%pb%p)%p

def gcd(a, b):
while b: a, b = b, a % b
return a

def pow(x, y, z):
res = 1
while y != 0:
if (y & 1) == 1:
res = (res * x) % z
y >>= 1
x = (x * x) % z
return res

关于线下的准备

菜刀

批量提交flag脚本

卡check机时间下服务

修改ssh密码：

  • Passwd-回车-输入旧密码-new
    修改数据库密码：
  • set password for mycms@localhost = password(‘123’);
  • mysql> set password for 用户名@localhost = password(‘新密码’);
  • mysqladmin -u用户名 -p旧密码 password 新密码

备份：

  • Find / -name index.php
  • tar -cvf /tmp/test.tar ./www/*
  • tar -cvf /home/option.tar ./www/*

备份数据库：

  • 找到配置文件 得到密码：find /var/www/html -path ‘config’

  • 找到mysqldump链接: find / -path ‘mysqldump‘

  • 备份：/usr/local/mysql/bin/mysqldump -uroot -pB7262a36a26c –all-databases >/tmp/all.sql

  • 还原：mysql -u root -p [dbnams] < /tmp/all.sql

  • 在mysql命令行还原：source ~/db.sql

  • mysql dump -uroot -proot –all-databases >/tmp/all.sql

  • mysql -uroot -p

  • mysqldump -u root -p -all-database > alldb.sql

连接ssh：

  • ssh 用户名@ip或主机名
  • ssh –p 23 ctf@ip
  • 获取信息：
  • id 用于显示用户的ID，以及所属群组的ID
  • root一般是0
  • 查看进程（中间件）：ps -aux
  • 确定对外端口：nmap -sV –open ip
  • 发现攻击目标：Nmap –sP 10.20.124.0/24
    备份/解压：
  • /tmp /home/xxx
  • zip –r /tmp/two.zip ./*
  • unzip xxx.zip –d 解压目录
  • tar -cvf /tmp/test.tar ./www/* 打包目录有www包
  • tar –xvf /tmp/test.tar –C /

抓取流量包：

  • sudo tcpdump -s 0 -w flow.pcap port 80

查找后门：

  • find . -name ‘*.php’ | xargs grep -n ‘eval(‘
  • find . -name ‘*.php’ | xargs grep -n ‘assert(‘
  • find . -name ‘*.php’ | xargs grep -n ‘system(‘
  • find . -name ‘*.php’ | xargs grep -n ‘func’
  • find . -name ‘*.php’ | xargs grep -n ‘]($_GET[‘
  • find . -name ‘*.php’ | xargs grep -n ‘`’
  • find . -name ‘*.js’ | xargs grep -n ‘function(req, res)’

查看最近被修改的五个文件 find . -type f | xargs ls -lt | head -5

find -name *.c -or -name *.h

find / -mmin -5 　　# 查找在系统中最后5分钟里修改过的文件(modify time)
find . -name ‘srm*’ 　　#表示当前目录下查找文件名开头是字符串‘srm’的文件

find . -name ‘.‘ | xargs grep -n ‘FLAG{‘
find . -name ‘.‘ | xargs grep -n ‘flag{‘
